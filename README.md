## using pyvisa

- install pyvisa: `python -m pip install pyvisa`
- install pyserial: `python -m pip install pyserial`
- install pyusb:`python -m pip install pyusb`
- install linux-gpib or gpib-ctypes `python -m pip install gpib-ctypes`
